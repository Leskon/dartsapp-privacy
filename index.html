Datenschutzerklärung für Simple Darts

Lokale Datenverarbeitung

Alle Spielerdaten (Namen, Statistiken, Match-Historie) werden ausschließlich lokal auf Ihrem Gerät gespeichert.

Diese Daten verlassen Ihr Gerät nur während aktiver Multiplayer-Sessions (siehe unten) und werden nicht dauerhaft auf Servern gespeichert.

Multiplayer-Funktion

Bei Nutzung der Online-Multiplayer-Funktion werden folgende Daten temporär übertragen:

  • Gewählter Spielername (frei wählbar, keine echten Namen erforderlich)
  • Spielstand und geworfene Punkte während der Session
  • 5-stelliger Session-Code zur Lobby-Verwaltung

Server und Datenspeicherung:

  • Daten werden über einen Server (darts.ddns.net) in Echtzeit übertragen
  • Alle Session-Daten werden nur temporär im Arbeitsspeicher gehalten
  • Nach Spielende oder bei Inaktivität werden alle Daten automatisch gelöscht
  • Es erfolgt keine dauerhafte Speicherung auf dem Server
  • Keine Erstellung von Benutzerkonten oder Profilen

Sichtbarkeit: Andere Spieler in derselben Session können Ihren gewählten Spielernamen und Spielstand sehen.

Keine Weitergabe an Dritte

Es erfolgt keine Weitergabe von Daten an Dritte, Werbepartner oder Analysedienste.

Diese App enthält keine Werbung und keine externen Tracker.

Ihre Rechte

Sie können die App jederzeit deinstallieren, um alle lokal gespeicherten Daten zu entfernen.

Bei Multiplayer-Sessions werden Server-Daten automatisch nach Spielende gelöscht.

Letzte Aktualisierung: 14.02.2026

Privacy Policy for Simple Darts

Local Data Processing

All player data (names, statistics, match history) is stored exclusively locally on your device.

This data only leaves your device during active multiplayer sessions (see below) and is not permanently stored on servers.

Multiplayer Feature

When using the online multiplayer feature, the following data is temporarily transmitted:

  • Chosen player name (freely selectable, no real names required)
  • Game score and thrown points during the session
  • 5-digit session code for lobby management

Server and Data Storage:

  • Data is transmitted in real-time via a server (darts.ddns.net)
  • All session data is only held temporarily in memory
  • After game end or inactivity, all data is automatically deleted
  • No permanent storage on the server
  • No creation of user accounts or profiles

Visibility: Other players in the same session can see your chosen player name and score.

No Third-Party Sharing

No data is shared with third parties, advertising partners or analytics services.

This app contains no advertising and no external trackers.

Your Rights

You can uninstall the app at any time to remove all locally stored data.

During multiplayer sessions, server data is automatically deleted after the game ends.
